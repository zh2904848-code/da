## 推荐实现方式

建议做成手机端 H5 互动页，再把链接放入微信公众号文章。

原因：

- 微信公众号正文里不能自由运行复杂 JavaScript。
- 复杂交互动画通常需要跳转到外部 H5 页面。
- H5 页面可支持点击、滑动、弹窗、镜头移动、时间轴动画等。

推荐技术：

- HTML + CSS + JavaScript
- GSAP 时间轴动画
- 必要时使用 Canvas / PixiJS
- 如果涉及 3D 或强空间感，可使用 Three.js

## 建议的交互结构

可以设计为：

1. 首屏显示手机聊天界面，引出事件。
2. 用户点击继续，镜头进入教室场景。
3. 横向移动或自动推进，依次看到旁观者、被欺凌者、围观同学。
4. 点击不同人物，弹出对应对话气泡。
5. 结尾出现问题，例如“遇到校园欺凌，你会怎么做？”
6. 用户选择后展示正确引导、倡议页或总结页。
